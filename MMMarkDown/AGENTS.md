# AGENTS

## 기본
- 모든 파일은 UTF-8 with BOM 으로 저장한다.
- 모든 커뮤니케이션은 한국어로 진행한다.

## 데이터/요약 규칙
- 노드 이름은 Markdown 파일명이며, 파일은 `MDDoc`에 저장한다.
- 상태는 `.mmm` JSON 파일로 관리한다.
- 앱 시작 시 `MDDoc`의 기존 `.md` 파일을 노드로 자동 추가한다.
- 요약은 한국어로 간결하게 3~5문장, 머리말 없이 내용만 출력한다.

## UI 표시 원칙
- 요약은 노드 카드 안에만 표시하고 별도 요약 패널은 사용하지 않는다.
- 노드 이름은 볼드로 표시한다.
- 노드 이동 시 애니메이션을 적용한다.

## 단축키
- `Insert`: 자식 노드 추가
- `Tab`: 형제 노드 추가
- `Shift` + `ArrowUp`: 현재 노드를 위로 이동
- `Shift` + `ArrowDown`: 현재 노드를 아래로 이동
- `Ctrl` + `X`: 노드 잘라내기
- `Ctrl` + `V`: 선택 노드 아래에 붙이기
- `F`: 선택 노드 포커스
- `Delete`: 노드 및 하위 노드/문서 삭제
- `F2` / `Enter`: VS Code로 열기

## 네비게이션
- 마우스 중클릭 드래그로 팬
- `Ctrl` + 마우스 휠로 줌 인/아웃
- 우하단 미니맵의 `+`, `-`, `Fit` 버튼 제공
- 미니맵 클릭으로 뷰포트 이동
- 축소 상태에서도 자유롭게 스크롤 가능해야 한다.

## 서버
- 코드 변경 시 자동 리로드가 기본 동작이다.
