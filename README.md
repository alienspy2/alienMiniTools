# alienMiniTools

- 내가 쓰려고대충 만들었지만, 남들도 쓸만한 툴들을 여기 모아 놓기
- 내가 쓰다가 발견한 버그만 잡은 정도. 제품화 되어있지 않다
- 코드 대부분은 AI 가 만든것

| 이름            | 설명                                | 상태    |
| ------------- | --------------------------------- | ----- |
| InvokeComfyUI | comfy ui 에서 만든 flow 를 외부에서 호출하는 툴 | 만드는 중 |
| MMMarkDown    | 로컬 마크다운 파일을 마인드맵으로 관리하는 웹 애플리케이션  | 사용 가능 |
| remoteKM      | 소프트웨어로 구현한 키보드/마우스 공유기            | 사용 가능 |
| MiniTCPTunnel | ssh 의 tunnel 기능만 떼어서 구현           | 만드는중  |
